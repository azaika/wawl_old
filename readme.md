# wawl
#### WinAPI C++ Wrapper Library
WinAPIをC++で薄めにラップしたライブラリです。  
ヘッダーオンリーですので適当に使ってください。  
msvc2013で動作確認済みです。

##### ライセンス
wawlはNYSL Version 0.9982のもとで公開されます。  
NYSLの詳細に関しては<http://www.kmonos.net/nysl/>を参照してください。
